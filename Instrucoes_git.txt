/*abaixo segue os comandos de como baixar o repositorio do github;
*primeiro deve-se baixar o git na maquina virtual com o seguinte comando:
*	apt-get install git
*depois configurar o git com seus dados;
*	git config --global user.name "seu_nome_usuario"
*	git config --global user.email "seu_email_usuario"
*para ajudar todos crie, se possivel seu repositorio no Documents do Ubuntu,
*vc usar esses comandos para isso:
*	cd Documentos/
*depois, dentro desse repositorio, baixe o projeto com o seguinte comando:
*	git clone git@github.com:luizerbetto/PUC_SOB_PROJ1.git
*talvez precise mudar o luizerbetto pelo seu proprio nome de usuario.
*com isso será criado um novo repositorio chamado PUC_SOB_PROJ1;
*quando quiser adicionar um novo item ao projeto use os seguintes comandos:
*	git add item_que_quer_mandar.tipo
*	git commit -m "mensagem sobre o que vai entrar no projeto"
*até o momento apesar desse comandos as mudanças não estão dentro do projeto,
para mandar deve usar o seguinte comando:
*	git push origin master
*todos os comandos usados foram tirados do seguinte site:
*	https://www.howtoforge.com/tutorial/install-git-and-github-on-ubuntu-14.04/
*/
